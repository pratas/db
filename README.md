<p align="left">
<b>Automatic construction of genome and chromosomal databases for metagenomics</b></p>

## Install
```
git clone https://github.com/pratas/dove.git
cd dove/
chmod +x BuildDB.sh
```

## Run
```
./BuildDB.sh
```
