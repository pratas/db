<p align="center">
<b>DOVE</b>: Automatic construction of genome and chromosomal databases for metagenomics</p>
<br>

## Install
<pre>
git clone https://github.com/pratas/dove.git
cd dove/
chmod +x BuildDB.sh
</pre>

## Run
<pre>
./BuildDB.sh
</pre>
